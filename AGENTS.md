# Workspace Instructions

- Toda alteracao feita no projeto deve terminar com uma nova build do executavel via `.\.venv\Scripts\python.exe .\build_exe.py`.
- O resultado final que precisa ficar atualizado e validado e `C:\Users\egidio\Desktop\Brasfoot\dist\Ultrafoot\Ultrafoot 26.exe`.
- Apos cada build, confirmar que o `dist\Ultrafoot\_internal\index.html` ou os assets alterados entraram no bundle e que o timestamp do `.exe` mudou.
- Se `dist\Ultrafoot\Ultrafoot 26.exe` estiver aberto, o Windows pode bloquear a troca do bundle; nesse caso, fechar o app e repetir a build antes de concluir a tarefa.
